Definición

Una Progressive Web App (PWA) es una tecnología que combina lo mejor de las aplicaciones móviles y las páginas web. Básicamente, una PWA utiliza las características de los navegadores web modernos para proporcionar una experiencia similar a la de una aplicación nativa en dispositivos móviles y de escritorio.
